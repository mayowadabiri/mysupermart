# MYSUPERMART

## Description

MySuperMart is an authentication API that assigns roles to 3 categories of users of a web platform - Supervisor, Clients, and Employee, and each user having specific API routes it can access

## Tools

1. NodeJS 
2. ExpressJS
3. MongoDB
